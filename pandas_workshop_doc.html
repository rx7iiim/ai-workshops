Introduction to Pandas 15 minutes

What is Pandas?

Pandas is a powerful Python library for data manipulation and analysis. It provides easy-to-use data structures and data analysis tools.

Key Features

üìä Data Structures

Series (1D) and DataFrame (2D) for handling structured data

üîß Data Manipulation

Easy filtering, grouping, merging, and reshaping

üìÅ File I/O

Read/write CSV, Excel, SQL, JSON, and more

üìà Analysis Tools

Statistical functions, time series, and visualization

Why Use Pandas?

  • Industry standard for data analysis in Python
  • Handles missing data gracefully
  • Intuitive syntax similar to SQL and spreadsheets
  • Integrates well with NumPy, Matplotlib, and Scikit-learn
  • Excellent performance for large datasets

Pandas Series 25 minutes

What is a Series?

A Series is a one-dimensional labeled array that can hold any data type (integers, strings, floats, Python objects, etc.).

Creating a Series

# From a list s1 = pd.Series([10, 20, 30, 40, 50]) print(s1) # From a list with custom index s2 = pd.Series([10, 20, 30], index=['a', 'b', 'c']) print(s2) # From a dictionary data = {'apple': 5, 'banana': 3, 'cherry': 8} s3 = pd.Series(data) print(s3) # From a NumPy array s4 = pd.Series(np.random.rand(5)) print(s4)
Output (s1): 0 10 1 20 2 30 3 40 4 50 dtype: int64

Series Attributes

s = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) print("Values:", s.values) # Array of values print("Index:", s.index) # Index object print("Data type:", s.dtype) # Data type print("Size:", s.size) # Number of elements print("Name:", s.name) # Series name (None by default)

Accessing Elements

s = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # By index label print(s['a']) # 10 print(s[['a', 'c']]) # Multiple elements # By position print(s[0]) # 10 print(s[0:2]) # First two elements # Boolean indexing print(s[s > 20]) # Elements greater than 20

Series Operations

s = pd.Series([1, 2, 3, 4, 5]) # Arithmetic operations print(s + 10) # Add 10 to each element print(s * 2) # Multiply each by 2 print(s ** 2) # Square each element # Statistical methods print("Mean:", s.mean()) print("Sum:", s.sum()) print("Max:", s.max()) print("Min:", s.min()) print("Standard deviation:", s.std()) # Other useful methods print("Unique values:", s.unique()) print("Value counts:", s.value_counts()) print("Is null?:", s.isnull())

Pandas DataFrames 40 minutes

What is a DataFrame?

A DataFrame is a 2-dimensional labeled data structure with columns that can be of different types. Think of it as a spreadsheet or SQL table.

Creating DataFrames

From a Dictionary

# Dictionary of lists data = { 'Name': ['Alice', 'Bob', 'Charlie', 'David'], 'Age': [25, 30, 35, 28], 'City': ['New York', 'Paris', 'London', 'Tokyo'], 'Salary': [50000, 60000, 75000, 55000] } df = pd.DataFrame(data) print(df)
Output: Name Age City Salary 0 Alice 25 New York 50000 1 Bob 30 Paris 60000 2 Charlie 35 London 75000 3 David 28 Tokyo 55000

From a List of Dictionaries

data = [ {'Name': 'Alice', 'Age': 25, 'City': 'New York'}, {'Name': 'Bob', 'Age': 30, 'City': 'Paris'}, {'Name': 'Charlie', 'Age': 35, 'City': 'London'} ] df = pd.DataFrame(data) print(df)

From CSV File

# Read CSV df = pd.read_csv('data.csv') # Read CSV with specific options df = pd.read_csv('data.csv', sep=',', # Delimiter header=0, # Row number to use as header index_col=0, # Column to use as index na_values=['NA']) # Additional strings to recognize as NaN

DataFrame Attributes

df = pd.DataFrame(data) print(df.shape) # Dimensions (rows, columns) print(df.columns) # Column names print(df.index) # Row indices print(df.dtypes) # Data types of each column print(df.info()) # Summary information print(df.describe()) # Statistical summary

Viewing Data

# First and last rows print(df.head()) # First 5 rows print(df.head(3)) # First 3 rows print(df.tail()) # Last 5 rows # Random sample print(df.sample(3)) # 3 random rows # Specific columns print(df[['Name', 'Age']]) # Column names print(df.columns.tolist())

Selecting Data

Selecting Columns

# Single column (returns Series) ages = df['Age'] # Multiple columns (returns DataFrame) subset = df[['Name', 'City']] # Using dot notation (only for valid Python identifiers) names = df.Name

Selecting Rows

# By position (iloc) print(df.iloc[0]) # First row print(df.iloc[0:3]) # First 3 rows print(df.iloc[[0, 2]]) # Rows 0 and 2 # By label (loc) df_indexed = df.set_index('Name') print(df_indexed.loc['Alice']) # Row with index 'Alice' print(df_indexed.loc['Alice':'Charlie']) # Range # Boolean indexing print(df[df['Age'] > 28]) print(df[df['City'] == 'Paris']) print(df[(df['Age'] > 25) & (df['Salary'] > 55000)])

Selecting Rows and Columns

# Using iloc (position-based) print(df.iloc[0:3, 1:3]) # Rows 0-2, columns 1-2 # Using loc (label-based) print(df.loc[0:2, 'Age':'City']) # Mixed selection print(df.loc[df['Age'] > 28, ['Name', 'Salary']])

Adding and Modifying Data

# Add new column df['Country'] = 'USA' df['Bonus'] = df['Salary'] * 0.1 # Modify existing column df['Age'] = df['Age'] + 1 # Add new row new_row = {'Name': 'Eve', 'Age': 27, 'City': 'Berlin', 'Salary': 52000} df = pd.concat([df, pd.DataFrame([new_row])], ignore_index=True) # Delete column df = df.drop('Country', axis=1) # Delete row df = df.drop(0, axis=0)

Data Operations 35 minutes

Sorting

# Sort by column df_sorted = df.sort_values('Age') df_sorted = df.sort_values('Age', ascending=False) # Sort by multiple columns df_sorted = df.sort_values(['City', 'Age']) # Sort by index df_sorted = df.sort_index()

Filtering

# Single condition young = df[df['Age'] < 30] # Multiple conditions (AND) result = df[(df['Age'] > 25) & (df['Salary'] > 50000)] # Multiple conditions (OR) result = df[(df['City'] == 'Paris') | (df['City'] == 'London')] # Using isin() cities = ['Paris', 'Tokyo'] result = df[df['City'].isin(cities)] # String methods result = df[df['Name'].str.startswith('A')] result = df[df['City'].str.contains('on')]

GroupBy Operations

data = { 'Department': ['IT', 'HR', 'IT', 'HR', 'IT', 'Sales'], 'Employee': ['Alice', 'Bob', 'Charlie', 'David', 'Eve', 'Frank'], 'Salary': [50000, 45000, 60000, 48000, 55000, 52000] } df = pd.DataFrame(data) # Group by single column grouped = df.groupby('Department')['Salary'].mean() print(grouped) # Multiple aggregations result = df.groupby('Department').agg({ 'Salary': ['mean', 'sum', 'count'], 'Employee': 'count' }) print(result) # Group by multiple columns df['Year'] = [2023, 2023, 2024, 2024, 2023, 2024] grouped = df.groupby(['Year', 'Department'])['Salary'].sum() print(grouped)

Merging and Joining

# Sample dataframes df1 = pd.DataFrame({ 'ID': [1, 2, 3], 'Name': ['Alice', 'Bob', 'Charlie'] }) df2 = pd.DataFrame({ 'ID': [1, 2, 4], 'Salary': [50000, 60000, 55000] }) # Inner join (default) merged = pd.merge(df1, df2, on='ID') # Left join merged = pd.merge(df1, df2, on='ID', how='left') # Right join merged = pd.merge(df1, df2, on='ID', how='right') # Outer join merged = pd.merge(df1, df2, on='ID', how='outer') # Concatenate vertically combined = pd.concat([df1, df2], ignore_index=True) # Concatenate horizontally combined = pd.concat([df1, df2], axis=1)

Pivot Tables

Data Cleaning 30 minutes

Handling Missing Data

data = { 'A': [1, 2, np.nan, 4], 'B': [5, np.nan, np.nan, 8], 'C': [9, 10, 11, 12] } df = pd.DataFrame(data) # Check for missing values print(df.isnull()) # Boolean DataFrame print(df.isnull().sum()) # Count per column print(df.isnull().sum().sum()) # Total count # Drop missing values df_dropped = df.dropna() # Drop rows with any NaN df_dropped = df.dropna(axis=1) # Drop columns with any NaN df_dropped = df.dropna(thresh=2) # Keep rows with at least 2 non-NaN # Fill missing values df_filled = df.fillna(0) # Fill with 0 df_filled = df.fillna(df.mean()) # Fill with column mean df_filled = df.fillna(method='ffill') # Forward fill df_filled = df.fillna(method='bfill') # Backward fill # Fill specific columns df['A'] = df['A'].fillna(df['A'].mean())

Handling Duplicates

data = { 'Name': ['Alice', 'Bob', 'Alice', 'Charlie'], 'Age': [25, 30, 25, 35] } df = pd.DataFrame(data) # Check for duplicates print(df.duplicated()) # Boolean Series print(df.duplicated().sum()) # Count # Drop duplicates df_unique = df.drop_duplicates() df_unique = df.drop_duplicates(subset=['Name']) # Based on specific column df_unique = df.drop_duplicates(keep='last') # Keep last occurrence

Data Type Conversion

df = pd.DataFrame({ 'A': ['1', '2', '3'], 'B': ['4.5', '5.6', '6.7'], 'C': ['2024-01-01', '2024-01-02', '2024-01-03'] }) # Convert types df['A'] = df['A'].astype(int) df['B'] = df['B'].astype(float) df['C'] = pd.to_datetime(df['C']) # Convert all at once df = df.astype({'A': int, 'B': float}) print(df.dtypes)

String Operations

df = pd.DataFrame({ 'Name': [' Alice ', 'BOB', 'charlie', 'David'] }) # String methods df['Name_clean'] = df['Name'].str.strip() # Remove whitespace df['Name_lower'] = df['Name'].str.lower() # Convert to lowercase df['Name_upper'] = df['Name'].str.upper() # Convert to uppercase df['Name_title'] = df['Name'].str.title() # Title case df['Name_len'] = df['Name'].str.len() # Length # String replacement df['Name'] = df['Name'].str.replace('BOB', 'Bob') # Extract patterns df['First_letter'] = df['Name'].str[0]

Renaming Columns

# Rename specific columns df = df.rename(columns={'A': 'Column_A', 'B': 'Column_B'}) # Rename all columns df.columns = ['col1', 'col2', 'col3'] # Clean column names df.columns = df.columns.str.strip().str.lower().str.replace(' ', '_')

Data Analysis 25 minutes

Statistical Analysis

df = pd.DataFrame({ 'A': [1, 2, 3, 4, 5], 'B': [10, 20, 30, 40, 50], 'C': [5, 15, 25, 35, 45] }) # Basic statistics print(df.describe()) # Summary statistics print(df.mean()) # Mean of each column print(df.median()) # Median print(df.std()) # Standard deviation print(df.var()) # Variance print(df.min()) # Minimum print(df.max()) # Maximum print(df.quantile(0.25)) # 25th percentile # Correlation print(df.corr()) # Correlation matrix print(df['A'].corr(df['B'])) # Correlation between two columns # Covariance print(df.cov())

Apply Functions

df = pd.DataFrame({ 'A': [1, 2, 3, 4, 5], 'B': [10, 20, 30, 40, 50] }) # Apply function to column df['A_squared'] = df['A'].apply(lambda x: x ** 2) # Apply function to row df['Sum'] = df.apply(lambda row: row['A'] + row['B'], axis=1) # Apply function to entire DataFrame df_doubled = df.apply(lambda x: x * 2) # Apply multiple functions result = df['A'].apply([np.mean, np.std, np.min, np.max])
